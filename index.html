Tugas Pendahuluan

1. Jelaskan apa yang dimaksud dengan HTML beserta fungsinya!
Jawaban:
HTML atau Hypertext Markup Language merupakan standar bahaa yang digunakan untuk mengelola atau mengature sebuah data dalam bentuk dokumen pada website. HTML dibuat oleh Seorang ahli fisika dengan Tim Bernes Lee pada lembaga penelitian CERN yang ada di Swiss. Beberapa fungsi HTML yaitu:

  • Mengontrol tampilan dari web page dan contentnya.
  • Mempublikasikan document secara online sehingga bisa diakses dari seluruh dunia.
  • Membuat online form yang bisa digunakan untuk menangani pendaftaran, transaksi secara online, dll.
  • Menambahkan objek-objek seperti image, audio, video dan juga java applet dalam document HTML.

2. Jelaskan perbedaan antara Tag, Atribut, Value dan Element pada HTML!
Jawaban:

  • Tag adalah sebuah penanda awalan dan akhiran dari sebuah element HTML.

    • Contoh: <p>, <h1>, <body>, dan sebagainya.

  • Atribut adalah kata khusus yang berada di dalam Tag pembuka. Atribut juga disebut sebagai modifier yang menentukan perilaku dari elemen

    • Contoh: <a>, <img>, <video>, dan sebagainya.

  • Value adalah nilai yang diberikan pada sebuah atribut. Value sendiri berfungsi untuk menentukan besar kecilnya dampak dari attribut dengan angka. Namun value sendiri tidak hanya berupa angka. Semua itu tergantung dari attribut yang digunakan.

    • Contoh: <p align="center"></p> dan sebagainya.

  • Element dalam HTML adalah sebuah komponen yang menyusun dokumen HTML. Elemen kadang juga disebut sebagai node, karena ia merupakan salah satu jenis node yang menyusun dokumen HTML dalam diagram HTML tree

    Contoh: <p align="center">Hello world!</p> dan sebagainya.

5. Tuliskan dan jelaskan atribut-atribut yang dapat digunakan pada Nomor 3!
Jawaban:

  • <a href = "index.html"> : Tag <a> mendefinisikan hyperlink. Sedangkan atribut "href" menentukan URL halaman yang di tuju.
  • <img src = "picture.png"> : Tag <img> tag digunakan untuk menyematkan gambar di halaman HTML. Sedangkan atribut "src" menentukan jalur ke gambar yang akan ditampilkan.
  • <p style = "color: white;">Paragraph<p> : Atribut "style" digunakan untuk menambahkan gaya ke elemen, seperti warna, font, ukuran, dan lainnya.

6. Tuliskan dan jelaskan atribut yang digunakan dalam menggabungkan baris dan kolom pada tabel!
Jawaban:

  • <td colspan="1"> : colspan adalah kependekan dari “column span” yang bisa mengartikan sebagai “berapa kotak kesamping”.
  • <td rowspan="3"> : rowspan yang berfungsi mengabungkan baris kebawah.
